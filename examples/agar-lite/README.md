# agar-lite

This is the current prototype example.

## Run

```bash
cd ../../server
npm install
./scripts/dev_lan.sh start
```

Open:
- `http://127.0.0.1:6868/`
- or the printed LAN URL on your phone.

## Quick bot test

- `/?bots=1&botCount=6`

## What this example demonstrates

- server-authoritative movement + smoothing
- pellets
- bots (per-room)
- minimap overlay
- robust input model (mouse/touch/keyboard)
