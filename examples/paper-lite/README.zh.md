# paper-lite（规划中）

中文 | [English](README.md)

目标：做一个简单的“跑圈 / 竞速 / 竞技场 runner” IO 规则集，基于同一套核心。

它用于验证框架能支持 Agar/Snake/Paper 之外的 IO 类型。

## 玩法概念（v1）

- 俯视角 2D 竞技场 + 环形赛道（或一组检查点）。
- 玩家用“加速 + 转向”，并限制转向速率。
- 目标选项（第一版先选一个）：
	1) **计时赛**：最快单圈 / 最佳用时
	2) **生存**：避开危险物，存活最久
	3) **得分**：吃拾取物并在检查点结算分数

## 规划中的规则

### 移动

- 服务端权威
- 速度上限、加速度、摩擦
- 转向速率限制（不能瞬间 180°）

### 赛道

- 用 polyline/waypoints 表达的环
- 检查点（A -> B -> C ... -> A）
- 圈数统计

### 碰撞

- 世界边界/墙（反弹或停止）
- 可选：玩家互撞

### 道具/危险物（可选）

- 加速带
- 减速区
- 地雷/陷阱

## 复用的核心能力

- rooms/sessions/reconnect/matchmaking
- 输入模型（转换成“期望转向 + 油门”）
- 小地图 + bots（按 waypoint 跑）
