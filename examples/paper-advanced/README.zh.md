# paper-advanced（规划中）

中文 | [English](README.md)

目标：做一个更完整的 Paper.io-like 规则集，基于同一套核心。

## 规划中的规则

### 实体

- 玩家（拥有一块“领地”）
- 尾迹（离开领地时产生）
- 领地网格/多边形表示

### 核心玩法

- 离开领地会拉出尾迹
- 回到领地会闭合环并占领圈内区域
- 碰撞规则：
  - 撞到任意尾迹（自己的或别人的）-> 死亡
  - 可选：头对头的判定规则

### 计分

- 领地面积
- 击杀数（可选）

### 平衡参数（config）

- 移动速度 / 转向速度
- 领地分辨率（网格大小）
- 重生保护时间

## 建议的实现拆分

- rooms/sessions/reconnect/input 放在 core 复用
- 实现一个 `rules/paper` 模块，负责领地/尾迹模拟
- 渲染：领地填充 + 尾迹 + 玩家 + 小地图
